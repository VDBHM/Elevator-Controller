This project implements a 3-floor Elevator Controller using Verilog HDL based on a Moore Finite State Machine (FSM) architecture. The system controls elevator movement between floors, manages door operations, and ensures safe transitions using synchronous sequential logic. Each floor and movement condition is represented as a distinct state, with outputs such as motor direction, current floor indication, and door status depending solely on the present state. The design includes an asynchronous reset for safe initialization and is verified using a comprehensive testbench that simulates various floor requests and generates waveform outputs for functional validation.This project implements a 3-floor Elevator Controller using Verilog HDL based on a Moore Finite State Machine (FSM) architecture.
